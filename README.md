A very basic file explorer which saves your recently opened tabs even after closing and reopening the file explorer. You can also reopen closed tabs in the file explorer with ctrl shift t. Someting i wanted for myself but maybe someone will find this funny to use.

Install Guide: Download release and run the exe: https://github.com/overpastdan/Pleiades/releases/download/v0.1.0/pleiades-explorer-windows.zip
